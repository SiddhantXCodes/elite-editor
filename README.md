# Elite Offline Editor

A lightweight browser-based HTML and Rich Text editor with live preview. Write HTML or plain text, edit it visually, and copy the generated HTML—all without leaving your browser.

## Screenshot

![Elite Offline Editor](assets/screenshot.png)

## Features

- ✍️ Write HTML or plain text
- 👀 Live WYSIWYG editing
- 📋 Copy generated HTML with one click
- 💾 Automatically saves your work using browser Local Storage
- 🔄 Content is restored automatically after page refresh
- 🌐 Works completely offline
- 🔒 Your data never leaves your browser

## How to Use

1. Type or paste HTML or plain text in the left editor.
2. Edit the rendered content visually in the right editor.
3. Click **Copy HTML** to copy the generated HTML.
4. Refresh the page anytime—your work will still be there.

## Technologies

- HTML5
- CSS3
- JavaScript
- TinyMCE
- Local Storage API
